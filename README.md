Numora is a basic and minimalist Android calculator application that has been created using the Kotlin language within Android Studio.

Features
Basic Arithmetic Operations (+, −, ×, ÷)
Minimalist UI
Supports backspace
Input appears as typed on screen in real time
Handles errors in case of invalid inputs
Efficient Performance


Tech Stack
Kotlin
Android Studio
XML (UI Design)
Android SDK


Workflow
Inputs numbers and operators through buttons
Expression gets evaluated as it appears on screen
Result obtained after pressing the "=" button
Backspace erases the most recently entered digit


Future Enhancements
Mode for Scientific calculator
Calculation history
Support for Dark Mode
Improved expression parsing capability


Installation
Download the code base from the repository
Open project in Android Studio
Build and run the app on emulator/physical device
